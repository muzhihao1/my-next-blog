# Scripts 目录说明

本目录包含项目的各种脚本工具。

## test-notion-connection.js

Notion API 连接测试脚本，用于验证 Notion 集成配置是否正确。

### 使用方法

```bash
# 确保已安装依赖
npm install

# 运行测试
npm run test:notion
```

### 测试内容

- API Token 有效性验证
- 数据库访问权限检查
- 数据库结构验证
- 内容获取测试

### 故障排除

如果测试失败，脚本会提供具体的错误信息和解决建议。常见错误：

1. **NOTION_TOKEN 未设置**：检查 `.env.local` 文件
2. **数据库未找到**：检查数据库 ID 或集成权限
3. **权限不足**：在 Notion 中邀请集成到数据库
